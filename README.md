# SIPAIM 2026 Official Website

Official website for the SIPAIM 2026 Conference - Processing of Images and Digital Health.

## About

SIPAIM (Procesamiento de Imágenes y Salud Digital) is a premier conference focused on medical imaging, digital health technologies, and biomedical engineering in Latin America.

## Features

- Responsive design with modern UI
- Interactive schedule with accordion-style agenda
- Speaker profiles
- Workshop registration system
- Tourism activity information

## Development

This is a static website built with HTML, CSS, and JavaScript.

### Prerequisites

- Python 3 (for development server)

### Installation

1. Clone the repository
2. No additional dependencies required

### Running Locally

Start the development server with Python:
```bash
python -m http.server 8000
```

The site will be available at `http://localhost:8000`

## Technologies Used

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- Vanilla JavaScript
- Bootstrap 5 (CDN)
- Google Fonts (Inter)
